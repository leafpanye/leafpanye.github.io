我的第一篇博客

发表于 2017-02-03 | 分类于 生活点滴 |

　　经过两天的折腾，终于把自己的博客搭建好了，有点小欣喜。

　　早就想弄一个博客，记录学习的知识和日常生活的点滴了。域名已经买了很久了，之前一直用的wordpress做后台，喜欢折腾一些网页的效果及插件，并且三天打鱼两天晒网。所以域名一直没派上什么用，买的空间也白白浪费了。

　　加上自己也不喜欢用wordpress写博客，觉得wp太臃肿了，写博客就应该简简单单的，没必要用到这么多的功能。所以，一开始的时候我是准备直接全手写html页面的，直到趁着过年放假有时间，折腾了一下，终于把我心目中博客该有的样子给折腾出来了，哈哈。
